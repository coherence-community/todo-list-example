How to Setup GraphiQL UI

To enable GraphiQL UI please see the README.md for instructions or carry out the following:

  1. Copy the contents in the sample index.html file from https://github.com/graphql/graphiql/blob/main/packages/graphiql/README.md into the file at 
    java/server/src/main/resources/web/graphiql.html
  2. Change the URL in the line 
    fetch('https://my/graphql' {
    to 
    http://127.0.0.1:7001/graphql
  3. Build an run the the project using the instructions in the README.md.
  4. Access the GraphiQL UI at 
    http://127.0.0.1/graphiql.html
